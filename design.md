# 시니어 일자리 추천 앱 디자인 계획

## 앱 개요
회원가입 없이 6단계 설문을 통해 시니어 맞춤 일자리 공고를 추천하는 모바일 앱.

---

## 컬러 팔레트

| 역할 | 색상 | 설명 |
|------|------|------|
| primary | `#2E7D9F` | 신뢰감 있는 청록 계열 (시니어 친화) |
| background | `#F8FAFC` | 밝고 깨끗한 배경 |
| surface | `#FFFFFF` | 카드 배경 |
| foreground | `#1A2B3C` | 주요 텍스트 |
| muted | `#6B7A8D` | 보조 텍스트 |
| accent | `#E8F4F8` | 선택 항목 하이라이트 배경 |
| border | `#D1DCE5` | 구분선 |
| success | `#27AE60` | 완료/활성화 상태 |
| inactive | `#C4CDD6` | 비활성 버튼 |

## 화면 목록

### 1. 스플래시 / 온보딩 화면 (WelcomeScreen)
- 앱 로고 + 앱 이름
- "시작하기" 버튼
- 간단한 앱 소개 문구

### 2. 설문 화면 (SurveyScreen)
- 상단 진행 바 (6단계 중 현재 단계 표시)
- 질문 텍스트 (큰 글씨, 시니어 가독성 고려)
- 선택지 카드 목록 (탭하면 선택/해제, 선택 시 primary 색상으로 강조)
- 하단 "다음" 버튼 (선택 전: 회색 비활성, 선택 후: primary 색상 활성화)
- Q1, Q2: 최대 2개 선택 가능 (멀티 셀렉트)
- Q3, Q4, Q5: 단일 선택
- Q6: 멀티 선택 가능, 선택 없이도 "다음" 활성화

### 3. 로딩 화면 (LoadingScreen)
- 추천 분석 중 애니메이션
- "맞춤 일자리를 찾고 있어요..." 텍스트

### 4. 추천 결과 화면 (ResultScreen)
- 상단: "회원님께 추천드리는 일자리" 헤더
- 좌우 스와이프 가능한 카드 슬라이더 (FlatList horizontal)
- 각 카드: 채용공고명, 모집직종, 근무요강, 추천 이유, 일치율
- 하단 인디케이터 (현재 카드 위치 표시)
- "다시 설문하기" 버튼

---

## 핵심 사용자 플로우

1. 앱 실행 → 웰컴 화면
2. "시작하기" 탭 → Q1 설문
3. 선택지 탭 → "다음" 버튼 활성화
4. "다음" 탭 → Q2 설문 (총 6단계 반복)
5. Q6 완료 후 "결과 보기" → 로딩 화면
6. 추천 결과 카드 슬라이드 → 좌우 스와이프로 공고 탐색
7. "다시 설문하기" → Q1으로 초기화

## 폰트 및 타이포그래피

- 질문 텍스트: 22px Bold (시니어 가독성)
- 선택지 텍스트: 18px Regular
- 버튼 텍스트: 18px SemiBold
- 카드 제목: 20px Bold
- 카드 본문: 16px Regular

---

## 레이아웃 특이사항

- 탭 바 없음 (단일 플로우 앱)
- 모든 터치 영역 최소 48px (시니어 접근성)
- 선택지 카드 간격 12px
- 카드 모서리 radius: 16px
